How this reads the machine, not the policy

  1. A benign, stateless UDP DNS query for a censored domain is sent toward China. The on-path GFW is bidirectional, so its own injector answers with a forgery — no node inside China needed.
  2. Fleet size comes from how many parallel injectors answer one query, and from the order they cycle their forged-IP pool (the Wallbleed model).
  3. Regional divergence — a province whose forged-IP pool disagrees with the national baseline is a candidate autonomous provincial firewall (the "wall behind a wall").
  4. An open-resolver fallback rides Chinese resolvers' outbound recursion, so the signal survives the decay of the inbound-injection channel.
